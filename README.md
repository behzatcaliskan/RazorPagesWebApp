# Razor Pages Web Uygulaması

Bu proje, Razor Pages yapısını öğrenmek ve ASP.NET Core üzerinde sayfa tabanlı web uygulaması geliştirme pratiği yapmak amacıyla hazırlanmıştır.

## Kullanılan Teknolojiler

- ASP.NET Core Razor Pages
- C#
- HTML
- CSS
- Bootstrap

## Proje Özellikleri

- Razor Pages sayfa yapısı
- PageModel mantığı
- HTML ve CSS ile arayüz düzenleme
- Temel sayfa yönlendirmeleri
- Form ve sayfa etkileşimleri

## Proje Amacı

Bu proje, ASP.NET Core MVC dışında Razor Pages yaklaşımını deneyimlemek ve sayfa tabanlı uygulama geliştirme mantığını öğrenmek için oluşturulmuştur.

## Teknik Kazanımlar

- Razor Pages dosya yapısı
- PageModel kullanımı
- HTML form yapısı
- ASP.NET Core sayfa yönlendirme mantığı
- Temel frontend entegrasyonu
